Architecture

Use PDF.js for parsing.
Use Pretext for text.

PDF.js has no plugin system. But its layered API means you can use the parser and canvas renderer while bringing your own text layer. That's exactly what pretext-pdf does — a 1,528-line wrapper, not a 128K-line fork.

Get started

Two lines to install.

// Install npm install pretext-pdf // Drop-in replacement for pdfjs-dist import { getDocument, TextLayer } from "pretext-pdf"; // TextLayer is now Pretext-powered. // Same API. Zero DOM reflows.
// Pinch-to-zoom PDF reader import { createPDFPinchReader } from "pretext-pdf/pinch"; const reader = createPDFPinchReader(container, { mode: "pinchMorph", }); await reader.open("document.pdf"); await reader.showPage(1);
